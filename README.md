# PySharp 2.0 Mini（阉割版）

**PySharp** 是一个由个人独立开发的轻量级跨语言 IDE，支持 Python 与 C# 的基础开发。该版本为开源“阉割版”，去除了核心插件系统与代码补全组件，仅保留基本的编辑器与运行环境，供学习、交流与参考使用。

## ✨ 特性 Features

- 支持 Python 和 C# 脚本运行
- 支持基本语法高亮与自动缩进

## 🚫 阉割说明（Lite 版本）

- ❌ 无代码补全
- ❌ 无主题编辑
- ✅ 可运行、可编辑、可保存

完整版请联系作者或关注后续发布信息。
